# Managing christophmediahub.com

## Project info

This is a professional media hub and content management system for Managing christophmediahub.com.

## Technologies

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Supabase (Planned migration to PHP)
- **Hosting**: Nakrotech (Planned)

## How to set up locally

1. Clone the repository.
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Project Structure

- `src/components`: UI components.
- `src/pages`: Main application pages.
- `src/integrations`: External service integrations (Supabase, etc.).
- `src/contexts`: React contexts for state management.
